* Overview
This repository contains a minimal, end-to-end Retrieval-Augmented Generation (RAG) proof of concept.

It demonstrates:
- Chunking and embedding an Org-mode document
- Storing vectors in Chroma
- Querying via a lightweight Python API
- Local LLM generation using Ollama

For more information, see my blog entry at: https://blog.achiwa.co.uk/posts/ragpoc/

* Architecture
#+BEGIN_SRC text
[Org README] → [Chunker] → [Embedding Model] → [Chroma DB]
                                   ↑
                                   │
                               [Query API]
                                   │
                                   ↓
                              [LLM Generator]
#+END_SRC

* Features
- Word-based chunking with overlap
- Embedding via Ollama (MiniLM or nomic-embed-text)
- Simple retrieval pipeline (top-k similarity)
- Minimal generation wrapper for local models

* Quick Start
- Setup Python 3.10+
- Git clone this project
#+begin_src 
git clone https://github.com/achiwa912/ragdemo.git
#+end_src
- Activate venv
#+begin_src 
cd gagdemo
python -m venv ve
source ve/bin/activate
#+end_src
- Install ChromaDB
#+begin_src 
pip install chromadb
#+end_src

- Install Ollama, and pull embedding and LLM models
#+begin_src 
curl -fsSL https://ollama.com/install.sh | sh
ollama pull all-minilm
ollama pull llama3.2:3b
ollama list # to verify
#+end_src

- Run the script
#+begin_src 
python ragdemo.py
#+end_src

After several minutes, you'll see the answer from Ollama:
#+begin_quote
The Shifting Learning Window method works by introducing a combination of high-frequency repetitions of small number of words and low-frequency repetitions of medium number of words. It uses a “learning window” that can have 10 words, initially filled with the first 10 words from a word book. As you repeat 10 words in the learning window, you memorize one or two words, and then the learning window replaces the memorized words with new ones. This process continues until the learning window reaches the end of the book, at which point it repeats the cycle.
#+end_quote
It took 4 minutes on an N150-based mini PC with 16GB memory.
